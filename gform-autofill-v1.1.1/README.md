# GForm AutoFill — Googleフォーム自動入力 Chrome拡張

Googleフォームの定型項目（名前・電話番号・住所など）を**ワンクリックで自動入力**するChrome拡張機能です。

---

## 主な機能

| 機能 | 詳細 |
|------|------|
| ワンクリック自動入力 | 拡張機能アイコンを押すだけでフォームを自動補完 |
| 姓名分離対応 | 「姓」「名」が別々の入力欄でも正しく入力 |
| 表記ゆれ対応 | 「電話番号」「携帯番号」「TEL」など異なるラベルでも認識 |
| 任意項目スキップ | 未登録の項目は自動的にスキップ |
| 住所自動結合 | 郵便番号・都道府県・市区町村・番地を結合して一行住所にも対応 |
| フリガナ対応 | 姓名フリガナ（カタカナ）も自動入力 |
| データのエクスポート/インポート | JSONファイルで設定をバックアップ・移行可能 |

---

## インストール手順

### 1. ZIPファイルを解凍する

ダウンロードした `gform-autofill.zip` を任意のフォルダに解凍してください。

### 2. Chromeの拡張機能管理ページを開く

Chromeのアドレスバーに以下を入力してEnterを押します：

```
chrome://extensions/
```

### 3. デベロッパーモードを有効にする

ページ右上の「**デベロッパーモード**」トグルをオンにします。

### 4. 拡張機能を読み込む

「**パッケージ化されていない拡張機能を読み込む**」ボタンをクリックし、解凍したフォルダ（`gform-autofill` フォルダ）を選択します。

### 5. 完了

Chromeのツールバーに 📋 アイコンが表示されれば完了です。

---

## 使い方

### Step 1: プロフィールを登録する

1. ツールバーの 📋 アイコンをクリック
2. 「⚙️ プロフィール設定」ボタンをクリック
3. 氏名・電話番号・住所などを入力
4. 「💾 保存する」ボタンをクリック

> **ポイント:** 全項目を入力する必要はありません。よく使う項目だけ入力すればOKです。未入力の項目はスキップされます。

### Step 2: Googleフォームを自動入力する

1. 入力したいGoogleフォームを開く
2. ツールバーの 📋 アイコンをクリック
3. 「⚡ ワンクリックで自動入力」ボタンをクリック
4. フォームの各項目が自動的に入力されます

## 対応しているフィールド

### 氏名

| フィールド | 対応するラベル例 |
|-----------|----------------|
| 姓 | 姓、苗字、名字、Last Name、Family Name |
| 名 | 名、名前、First Name、Given Name |
| 氏名（フルネーム） | 氏名、お名前、Full Name、Name |
| 姓（フリガナ） | フリガナ（姓）、セイ |
| 名（フリガナ） | フリガナ（名）、メイ |
| 氏名（フリガナ） | フリガナ、ふりがな、よみがな |

### 連絡先

| フィールド | 対応するラベル例 |
|-----------|----------------|
| 電話番号 | 電話番号、電話、携帯番号、携帯電話、TEL、Phone、Mobile |
| メールアドレス | メール、メールアドレス、Email、E-mail |

### 住所

| フィールド | 対応するラベル例 |
|-----------|----------------|
| 郵便番号 | 郵便番号、〒、Postal Code、Zip Code |
| 都道府県 | 都道府県、Prefecture |
| 市区町村 | 市区町村、市町村、City |
| 番地・建物名 | 番地、建物名、Street Address |
| 住所（全体） | 住所、ご住所、Address |

### その他

| フィールド | 対応するラベル例 |
|-----------|----------------|
| 所属・会社名 | 所属、会社名、学校名、Organization、Company |
| 部署・学部 | 部署、学部、学科、Department |
| 年齢 | 年齢、Age |
| 生年月日 | 生年月日、誕生日、Birthday |
| 性別 | 性別、Gender |

## プライバシーについて

- 登録した個人情報は **Chromeブラウザのローカルストレージ（chrome.storage.sync）** に保存されます
- 外部サーバーへの送信は一切行いません
- データはChromeアカウントでログインしている場合、他のデバイスと同期されます（Chrome Syncの設定による）

---

## 注意事項

- Googleフォームのページ（`https://docs.google.com/forms/`）でのみ動作します
- フォームのデザインや構造によっては、一部の項目が自動入力されない場合があります
- ラジオボタン・チェックボックス・プルダウンメニューは現バージョンでは対応していません（テキスト入力のみ対応）
- 自動入力後は必ず内容を確認してから送信してください
