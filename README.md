# Chrome拡張機能：AIフォーム自動入力 (AI Auto Form Input)

Web上のあらゆるフォームに対して、項目（氏名、住所、メールアドレスなど）をAIが自動で認識し、あらかじめ設定した内容をワンクリックで入力できるChrome拡張機能です。

## 背景・課題

Webサービスへの登録やECサイトでの購入時など、私たちは何度も同じようなフォームに個人情報を入力しています。ブラウザの自動入力機能は便利ですが、対応していないサイトや、項目のid/nameが特殊なフォームでは機能しないことが多々あります。

この定型的な作業を、AIの力で完全に自動化できないかと考えました。「どんな形式のフォームでも、人間のように『ここには名前を入れるべきだ』と判断してくれる」ツールがあれば、日々の小さなストレスを解消し、生産性を大幅に向上させられるはずです。このアイデアの実現可能性と実用性を確かめるため、本拡張機能を開発しました。

## 機能

- **AIによる項目認識:** 表示されているフォームのHTML構造とラベルテキストをGPT-4 APIに送信し、それぞれの入力欄が何を求めているか（例：「氏名」「メールアドレス」「郵便番号」など）をAIが判断します。
- **ワンクリック自動入力:** 拡張機能のアイコンをクリックするだけで、AIが認識した全ての項目に、あらかじめ設定しておいた個人情報を自動で入力します。

## 技術スタック

- **言語:** JavaScript, HTML
- **API:** OpenAI API (GPT-4)

## こだわった点・工夫

このプロジェクトの核心は、**「AIにWebページの文脈を理解させる」**という点にあります。単なるキーワードマッチングではなく、GPT-4の能力を最大限に活用して、人間のような柔軟な判断を実現することを目指しました。

- **プロンプトエンジニアリング:** AIにフォームの構造を的確に伝え、高い精度で項目を特定させるためのプロンプト設計に最も時間をかけました。「バイブス・コーディング」の真骨頂であり、どうすればAIが最高のパフォーマンスを発揮してくれるかを試行錯誤するプロセスそのものが、このプロジェクトの面白さでした。
- **実用性へのこだわり:** すぐに自分自身が使えるツールとして開発したため、いかにシンプルで直感的な操作性を実現できるかを重視しました。

## 今後の展望

- 入力内容を複数パターン保存し、用途（仕事用、プライベート用など）に応じて切り替えられる機能。
- セキュリティ強化のため、個人情報をローカルストレージに暗号化して保存する機能。
- チェックボックスやラジオボタン、ドロップダウンメニューへの対応。
